## About

This is my emacs setup, which I use in my every day workflow. It is continuously being tweaked and updated as I refine the way I work.

## Installation

``` bash
$ cd ~
$ git clone https://github.com/gchp/emacs-config.git ./.emacs.d
```

## Included packages

- Magit
- Flatland Theme
- Yasnippet
- Git Gutter
- JS2
- Smartparens
- Key Chord
- icomplete+
- Web Mode
